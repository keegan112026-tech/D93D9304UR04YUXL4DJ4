# 監察院糾正案報告  
- **監察院糾正報告 (114社正0005)**：監察院於2025年5月29日通過關於「剴剴案」的糾正案，指出衛福部、新北市府及臺北市府在本案中未善盡兒少保護責任。報告認定衛福部長期未有效監督收出養媒合服務、相關法規未檢討修正；新北市政府依賴兒盟社工評估、未整合資源與訪視；臺北市政府對居家托育輔導流於形式，導致三方協力機制失靈、錯失發現兒童受虐契機【3†L4-L8】【12†L108-L110】。監察院已要求各機關改善上述疏失並強化法制規範。  

# 衛福部與雙北市政府政策檢討／新聞稿  
- **衛生福利部新聞稿 (2025年5月29日)**：衛福部發布聲明表示，收到監察院糾正報告後將「全面檢視、持續精進」。具體措施包括：修正收出養必要性評估權責，自113年5月起由地方政府主導，並於114年3月28日頒布評估報告格式範本【27†L144-L152】；落實居家托育訪視規範，依2024年6月3日修正之指引採「不預約、不同時段、親見兒童」的訪視方式，並增加對全日托或夜托嬰幼兒的訪視頻次【27†L151-L159】；提升訪視輔導員待遇與人力配置，並將兒虐辨識納為訪視員及托育人員的常態訓練課程【27†L155-L159】。衛福部並指出，兒童收出養管理辦法已進入法制作業階段，預計於2025年夏季完成修法，將進一步強化制度監督和保障。  

- **臺北市政府社會局新聞稿 (2025年5月29日)**：臺北市社會局對監察院記者會質疑作出澄清，強調居家托育中心訪視依規定落實。報告指出，剴剴在臺北市劉姓保母處於全日托期間，訪視員於9月26日（一個月內）完成首次無預警訪視，並於托育地點實際見到兒童，當時並無發現異常；原定12月下旬再訪，但案發前曾安排在此時段執行【35†L198-L201】。對於監察院提及的瘀傷照片一事，北市府表示相片來自兒盟社工於10月24日提供家屬，其具體拍攝時間難以查證，已將相關情資提交司法機關調查【35†L210-L212】。新聞稿最後強調，臺北市將持續優化制度，包括與臺灣北北基（臺北市、新北市、桃園市）社福單位協作建立「三方共訪機制」、加強訪視員及保母專業訓練等，以確保兒童在安全環境中成長【35†L214-L218】【38†L203-L210】。  

- **臺北市政府社會局新聞稿 (2025年5月13日)**：針對劉姓保母姊妹在本案的刑事宣判，臺北市社會局發布新聞稿指出，市府支持司法從重量刑，並已提出多項「剴剴案精進計畫」。內容包括：自113年底起推動「7+1改革守護計畫」，具體措施如加強保母在職訓練並增開回訓課程、提高全日托兒童訪視頻率至每年6次、引入脆弱家庭兒童三方共訪機制等【38†L203-L210】；同時，社會局強調訪視員「敏感度訓練」為114年重點課程，並將持續滾動檢討托育訪視機制，以避免類似慘劇再次發生【38†L209-L210】【38†L219-L220】。新聞稿提及，部分制度如「每年6訪、三方共訪」尚待中央立法或制度決議完成，一旦法規修訂將立即配合執行【38†L209-L210】【38†L219-L220】。  

# 立法院修法進度（「剴剴條款」）  
- **刑法修正案（俗稱「剴剴條款」）**：立法院於2025年7月18日三讀通過刑法修正案，增訂第272條之1並修正第286條【22†L1-L4】。修正後規定，對未滿7歲兒童犯故意殺人罪者加重其刑，若以凌虐方式殺害7歲以下幼童，處死刑或無期徒刑【22†L1-L4】；對未滿7歲之人施以凌虐致死者處死刑、無期徒刑或10年以上有期徒刑【22†L1-L4】。修正案已於2025年8月1日由總統公布生效，這些加重刑責的立法即是社會俗稱的「剴剴條款」，目的是對重大兒虐行為予以嚴懲。
